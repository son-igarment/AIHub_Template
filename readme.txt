A starter theme for AIHub with a refreshed palette and no bundled images.

== Description ==

AIHub Template is a lightweight, developer-friendly starting point with a custom deep blue and coral color system. All image assets have been removed so the theme can be branded entirely with code and typography.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and choose the theme .zip file. Click Install Now.
3. Click Activate to start using the theme.
